Title: Customer Scheduler Application
Purpose: to add customers to a database and schedule, modify and delete appointments with those customer.

Author: Brian Avalos
Email: baval10@wgu.edu

IDE: IntelliJ IDEA Community Edition 2022.3.1
JDK: Java SE Development Kit 17.0.5
JavaFX: 17.0.2
MySQL Connector: mysql-connector-java-8.0.13

Directions: To run the application you must have a username and password to log in.
after you log in you can add customers by clicking the add button under the customer table.
Once you have customers stored in the database you can then add an appointment with that customer.
After you have customers and appointments you can then add more or modify and delete existing customers and appointments.
